Return predicted duration in hours (float) | **Pure PHP matrix operations** (no external library needed for 4×4): <br>```php <br>// Matrix multiply A × B <br>private function matMul(array $A, array $B): array {...} <br>// Matrix transpose <br>private function matTranspose(array $M): array {...} <br>// Gaussian elimination inverse <br>private function matInverse(array $M): array {...} <br>``` <br>**Discrete solution of GM(1,N):** <br>`X1_hat(k+1) = (X1(0)(1) - (b2*X2_ago(k+1) + b3*X3_ago(k+1) + b4*X4_ago(k+1))/a) * e^(-a*k) + (b2*X2_ago(k+1)+b3*X3_ago(k+1)+b4*X4_ago(k+1))/a` <br>**Note:** Fix thesis typo — implement strictly as **GM(1,4)** not GM(2,N) | High |

## GM(1,4) Algorithm Quick Reference (for `GmPredictionService.php`)

```
Variables:
  X1 = waktu selesai aktual (jam) — TARGET variable
  X2 = berat cucian (kg)
  X3 = complexity_score layanan (integer 1–5)
  X4 = kapasitas_mesin (% utilisasi, 0–100)

Steps:
1. AGO:       X_i^(1)(k) = Σ X_i^(0)(j) for j=1..k
2. Mean Z:    Z1^(1)(k)  = 0.5 * [X1^(1)(k) + X1^(1)(k-1)]
3. Matrix B:  row k = [-Z1^(1)(k), X2^(1)(k), X3^(1)(k), X4^(1)(k)]
4. Vector Y:  Y[k] = X1^(0)(k)
5. Params:    θ = (B'B)^-1 * B'Y  →  a=θ[0], b2=θ[1], b3=θ[2], b4=θ[3]
6. Predict:   X1_hat^(1)(k) = [X1^(0)(1) - Σbi*Xi^(1)(k)/a] * e^(-a*(k-1)) + Σbi*Xi^(1)(k)/a
7. IAGO:      X1^(0)(n+1) = X1_hat^(1)(n+1) - X1_hat^(1)(n)

Output: predicted duration in hours (float)
Note: STRICTLY GM(1,4) — NOT GM(2,N) (fix thesis Chapter 3 framework typo)
```

## Accuracy Metrics Reference

| Metric | Formula | Threshold (Lewis 1982) |
|--------|---------|----------------------|
| MAPE | `(1/n) × Σ\|Xt - X̂t\|/Xt × 100%` | < 10% = Sangat Baik, 10–20% = Baik, 20–50% = Layak, > 50% = Buruk |
| MAE | `(1/n) × Σ\|Xt - X̂t\|` | Lower = better (in hours) |
